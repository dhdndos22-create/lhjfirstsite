# 피싱월드 안전 장비 합성 시스템

## 이번 버전의 핵심

- 캐릭터를 화면에서 레이어로 직접 겹치지 않습니다.
- Canvas에서 먼저 완성 캐릭터 PNG 개체 1개를 생성합니다.
- 로비에는 완성 결과 한 장만 출력합니다.
- 캐릭터 크기는 이전 큰 버전의 약 70%입니다.
- 장비 이미지가 1024×1536 규격과 다르면 자동으로 제외합니다.
- 장비 로딩 또는 합성에 실패해도 마지막 정상 캐릭터로 복구합니다.
- 프로필이나 낚시 화면에도 동일한 캐릭터 개체를 재사용할 수 있습니다.

## 중요한 현재 상태

현재 메인 캐릭터 1 이미지는 모자, 조끼, 바지, 신발, 낚싯대를 포함한
완성 이미지입니다. 따라서 지금은 정상 출력과 시스템 안전성이 먼저 적용되어 있습니다.

추후 실제 교체형 장비를 사용하려면 장비가 제거된 clean base를
1024×1536 규격으로 한 번 제작해야 합니다. clean base가 준비되면
`js/character-composer.js`의 base 이미지 경로만 교체하고,
신규 장비 PNG를 카탈로그에 등록하면 됩니다.
