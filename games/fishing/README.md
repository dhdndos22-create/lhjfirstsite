# 피싱월드 시작 화면

## 효종월드 로그인 연결

- 효종월드 메인 로그인 성공 시 저장되는 `localStorage.hyojongUser`를 읽습니다.
- 로그인하지 않은 상태로 게임 시작 버튼을 누르면 효종월드 메인으로 이동합니다.
- 로그인 상태에서는 `fishingSession.username`에 유저명이 저장됩니다.
- 다른 탭에서 로그아웃하면 피싱월드도 메인으로 이동합니다.

현재 단계는 기존 효종월드 로그인 상태와 피싱월드를 연결한 것입니다.
랭킹과 친구 시스템을 안전하게 운영하려면 이후 Supabase Auth 기반 사용자 ID 연결이 필요합니다.
