# 피싱월드 공통 패널 UI 버전

## 적용 내용

상점, 인벤토리, 장비, 뽑기, 업적, 우편이 하나의 공통 패널 화면을 사용합니다.

공통 화면에 포함되는 항목:

- 상점 배경 이미지
- 상단 레벨 / 골드 / 에너지 UI
- 반투명 남색 패널 이미지
- 화면 제목
- 선택형 카테고리 탭
- 기능별 콘텐츠 슬롯
- 선택형 페이지 이동 영역
- 오른쪽 위 공통 뒤로가기 버튼

상품카드, 인벤토리 카드, 장비 카드는 공통 UI와 분리되어 있습니다.
각 기능은 `#commonPanelBody` 슬롯에 독립 컴포넌트를 렌더링하면 됩니다.

## 주요 파일

- `js/common-panel-ui.js`
  - 공통 화면 정의
  - 카테고리 탭 생성
  - 콘텐츠 슬롯 관리
  - 페이지 이동 UI 관리
- `css/common-panel-ui.css`
  - 공통 패널 위치와 디자인
  - 뒤로가기 버튼 및 카테고리 스타일
- `js/fishing-world.js`
  - 로비 메뉴와 공통 패널 연결
  - 플레이어 상태 UI 공통 출력

## 새 메뉴 추가 예시

`js/common-panel-ui.js`의 `PANEL_DEFINITIONS`에 항목을 추가하고,
로비 메뉴 버튼의 `data-menu` 값을 같은 키로 지정하면 됩니다.
