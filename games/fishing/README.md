# 피싱월드 완성 캐릭터 개체 합성 버전

## 이번 변경

- HTML에서 여러 캐릭터 레이어를 직접 겹쳐 출력하던 방식을 제거했습니다.
- 숨겨진 Canvas에서 장비 레이어들을 먼저 하나의 PNG 캐릭터 개체로 합성합니다.
- 로비에는 합성 결과 이미지 1개만 출력됩니다.
- 장비를 변경하면 완성 캐릭터 개체를 다시 생성합니다.
- 향후 프로필, 낚시 화면도 같은 완성 캐릭터 개체를 등록해서 재사용할 수 있습니다.
- 상단 레벨/골드/에너지 UI를 이전보다 약 30% 크게 보이도록 조정했습니다.

## 주요 파일

- `js/character-composer.js`
  - 캐릭터 Canvas 합성
  - 이미지 캐시
  - 완성 Blob URL 생성
  - 여러 화면의 캐릭터 이미지 동시 갱신
- `CHARACTER-ASSET-RULES.md`
  - 신규 장비 PNG 제작 규칙
- `images/player/character-standard.json`
  - 장비 이미지 공통 규격

## 신규 장비 규격

- 1024 × 1536
- 투명 PNG
- 정면
- 중심 X = 512
- 발바닥 Y = 1450
- 부위만 잘라내지 않고 전체 캔버스 유지
- 기본 캐릭터와 같은 좌표
