# 피싱월드 - 새 상태 UI 적용 버전

적용 내용:
- 상단 레벨/경험치 UI 교체
- 상단 골드 UI 교체
- 상단 에너지 UI 교체
- 이미지에는 숫자가 포함되지 않으며 HTML 텍스트로 실시간 표시
- 로비와 상점 화면에 동일한 상태 UI 사용
- 최초 기본값: 레벨 1, 골드 0, 에너지 10
- 최대 레벨 100, 에너지 보유 제한 없음

## 2026-07 상점 화면 레이어 수정
- `images/shop-background.jpeg`: 상점 전용 배경
- `images/shop-panel-overlay.png`: 남색 상점 UI 레이어
- 상점 화면에서도 레벨/골드/에너지 UI 유지
- 남색 UI는 상단 상태 UI 아래부터 배치되도록 CSS에서 크기와 위치 조정

## 상점 상품 시스템
- 카테고리: 장비 / 소모품 / 제화
- 한 페이지 상품 카드: 가로 2 × 세로 4 = 8개
- 8개 초과 시 이전/다음 페이지 버튼 사용
- 제화 카테고리에 에너지 상품 10종 예시 포함
- 상품 이미지 칸은 빈 자리로 구성
- 카드 또는 구매 버튼 클릭 시 상품 정보 팝업 표시
- 구매 시 골드 차감 및 에너지 지급


## 상품 카드 레이아웃 변경
- 모바일에서 카드 크기를 키우기 위해 1페이지 4개(세로 4개)로 변경
- 카드 왼쪽: 정사각형 상품 이미지 영역
- 카드 중앙: 상품명과 가격
- 카드 오른쪽: 구매/판매 버튼
- 상품이 4개를 넘으면 다음 페이지로 이동
