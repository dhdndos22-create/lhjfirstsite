# 피싱월드 레이어 캐릭터 + 장비 변경 적용본

## 적용 내용
- 로비 중앙에 `메인 캐릭터 1` 표시
- 캐릭터를 기본 몸/머리/상의/하의/신발/낚싯대/앞쪽 손 레이어로 구성
- 캐릭터 또는 하단 `메뉴` 버튼을 누르면 장비 변경 패널 표시
- 장비 부위: 머리, 상의, 하의, 신발, 낚싯대
- 현재 첫 장비 세트와 각 부위의 `착용 안 함` 선택 가능
- 선택 장비는 로그인 유저명 기준 localStorage에 저장
- 상태 UI 및 버튼의 물방울 효과 유지

## 참고
첫 장비 이미지는 콘셉트 시트에서 분리해 정렬한 초기 버전입니다.
추후 같은 캐릭터 자세와 좌표로 제작한 투명 PNG로 교체하면 코드 변경 없이 품질을 높일 수 있습니다.
