# 피싱월드 시작 화면

현재 포함된 기능

- 버튼이 제거된 피싱월드 시작 배경 이미지
- 투명 배경 게임 시작 버튼 이미지
- 두 이미지를 서로 다른 레이어로 배치
- 게임 시작 버튼 영역만 클릭 가능
- 버튼 누름 애니메이션
- 작은 거품이 올라오는 클릭 효과
- 모바일 브라우저 화면을 위아래 빈 공간 없이 꽉 채움
- 주소창 표시 상태가 바뀌어도 실제 화면 높이에 맞게 자동 보정
- 로비 및 실제 게임 화면은 아직 없음

## 실행 방법

`index.html`을 브라우저에서 열거나 VS Code의 Live Server로 실행하세요.

## 파일 구조

```text
fishing/
├─ index.html
├─ css/
│  └─ fishing-world.css
├─ js/
│  └─ fishing-world.js
└─ images/
   ├─ start-background.png
   └─ start-button.png
```

## 화면 채우기 방식

- 게임 영역은 모바일의 실제 보이는 화면 전체를 사용합니다.
- 배경 이미지는 `object-fit: cover`로 표시됩니다.
- 휴대폰 화면 비율에 따라 배경 이미지의 가장자리 일부는 조금 잘릴 수 있습니다.
- 중요한 제목과 버튼은 이미지 제작 단계에서 중앙 안전 영역에 배치해야 합니다.

## 버튼 위치 조절

`css/fishing-world.css`의 `.start-button`에서 아래 값을 조절하면 됩니다.

- `top`: 버튼의 세로 위치
- `width`: 버튼 크기
