피싱월드 UI 초기 버전

구성
- fishing.html
- css/fishing.css
- js/main.js

실행
1. 폴더를 VS Code로 엽니다.
2. fishing.html을 Live Server로 실행합니다.
3. 게임 시작 버튼을 누르면 로비로 전환됩니다.

현재 구현
- 피싱월드 시작 화면
- 시작 화면 물결/배/찌 애니메이션
- 로비 화면
- 닉네임, 레벨, 경험치, 골드, 보석 UI
- STAGE 1 동네 연못 카드
- 현재 낚싯대와 미끼 표시
- 장비, 가방, 도감, 스테이지, 뽑기 메뉴
- 준비 중 팝업
- 사이드 메뉴
- 모바일 반응형

다음 개발 권장 순서
1. 스테이지 선택 화면
2. 장비 인벤토리
3. 물고기 데이터와 등급
4. 실제 낚시 플레이 화면
