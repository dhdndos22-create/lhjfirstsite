# 캐릭터 장비 레이어 제작 규칙

- 형식: PNG
- 캔버스: 1024 × 1536
- 배경: 완전 투명
- 중심선: X = 512
- 발바닥 기준선: Y = 1450
- 장비 부분만 그리고 나머지는 투명
- 장비 크기만큼 잘라내지 않음
- 기본 캐릭터의 크기와 위치를 변경하지 않음
- 모든 레이어는 Canvas의 (0, 0)에 그대로 합성
- 코드에서 개별 좌표, 배율, 보정값을 사용하지 않음

레이어 순서:

1. base
2. bottom
3. shoes
4. top
5. rodBack
6. bodyFront
7. rodFront
8. head

규격이 맞지 않는 이미지는 코드가 자동으로 제외하여 캐릭터 깨짐을 방지합니다.
